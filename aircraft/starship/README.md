# Starship AAO touch panels

Aircraft-specific files for the unified `bksq-panels` package. Copy the whole package root, not this folder by itself.

## AAO script list

Import `bksq-starship-scripts.xml` in AAO for the `bksq-starship-panel` group.

## Pages

Open pages through the package root `index.html` or directly under `pages/`:

- `pages/overhead.html`
- `pages/ecs.html`
- `pages/antiice.html`
- `pages/autopilot.html`
- `pages/mfd.html`
- `pages/misc.html`
- `pages/pedestal.html`
- `pages/tests.html`
